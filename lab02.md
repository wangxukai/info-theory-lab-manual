# 实验2、熵函数曲线绘制  
## 一、实验内容  
分别用Excel和Matlab软件绘制二进熵函数曲线。在Matlab软件中绘制三进熵函数曲面。
## 二、实验环境
1. 计算机
2. Windows 2000 或以上
3. Microsoft Office XP 或以上
4. Matlab 6.5或以上  
## 三、实验目的
1. 掌握Excel的数据填充、公式运算和图表制作
2. 掌握Matlab绘图函数
3. 掌握、理解熵函数表达式及其性质
